About Fira Code

Discover the story behind the world's favorite programming font

About Us

Fira Code is a free monospaced font containing ligatures for common programming multi-character combinations. This project was born from the desire to make code more readable and beautiful. The font is an extension of the Fira Mono font, which was created by Mozilla's Carrois Type Design team.

Fira Code has become one of the most popular programming fonts in the developer community, with support for over 600 ligatures and continuous improvements based on community feedback. The font is designed to enhance the coding experience across all platforms and code editors, including special variants like FiraCode Nerd Font for VSCode and other modern editors.

Our Mission

Our mission is to improve the daily experience of developers worldwide by providing a beautiful, functional, and free programming font. We believe that code should not only work well but also look great. By offering Fira Code font with programming ligatures, we aim to reduce visual clutter in code and help developers focus on what matters most - writing great software.

We are committed to maintaining Fira Code as a free and open-source project, ensuring that every developer, regardless of their background or resources, has access to professional-quality typography for their coding work.

Our Vision

We envision a future where every developer has access to tools that enhance their productivity and enjoyment of coding. Fira Code is more than just a font - it's part of a movement to make programming more accessible, enjoyable, and visually appealing.

Our vision includes expanding support for more programming languages, creating specialized variants like the FiraCode Nerd Font for VSCode, and fostering a community of developers who value both functionality and aesthetics in their development environment. We strive to set the standard for what a modern programming font should be.

Why Fira Code?
