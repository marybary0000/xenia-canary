PROJECT OVERVIEW
--------------------------------------------------------------------------------
This is a professional, responsive, multi-page website for Fira Code font.
The website is built using HTML, CSS, and JavaScript with a clean, modern design.

FILE STRUCTURE
--------------------------------------------------------------------------------
ABC/
├── index.html          - Home page with hero section and features
├── about.html          - About page with mission, vision, and details
├── contact.html        - Contact page with Google Form integration
├── download.html       - Download page with installation instructions
├── style.css           - Main stylesheet for all pages
├── script.js           - JavaScript for interactivity and animations
└── README.txt          - This file (instructions and documentation)

HOW TO RUN THE WEBSITE
--------------------------------------------------------------------------------
1. OPENING THE WEBSITE:
   - Simply double-click on "index.html" to open it in your default browser
   - Alternatively, right-click on any HTML file and select "Open with" your 
     preferred browser (Chrome, Firefox, Edge, Safari, etc.)

2. NAVIGATING THE WEBSITE:
   - Use the navigation menu at the top to switch between pages
   - Click on links and buttons to explore different sections
   - The website is fully responsive - try resizing your browser window!

3. TESTING ON MOBILE:
   - Open developer tools in your browser (F12 or Ctrl+Shift+I)
   - Click the device toolbar icon (Ctrl+Shift+M)
   - Select different device sizes to test mobile responsiveness

HOW TO EDIT THE WEBSITE
--------------------------------------------------------------------------------
1. EDITING CONTENT:
   - Open any HTML file in a text editor (Notepad++, VS Code, Sublime Text, etc.)
   - Find the section you want to edit
   - Modify the text between HTML tags
   - Save the file and refresh your browser to see changes

2. CHANGING COLORS:
   - Open "style.css" in a text editor
   - Find the ":root" section at the top (around lines 9-16)
   - Modify the color values:
     --primary-color: #F7F6F1    (Main background color)
     --secondary-color: #EEEEEE  (Section backgrounds)
     --accent-color: #FFFFFF     (Card backgrounds)
     --button-primary: #5755d9   (Button color)
   - Save and refresh to see changes

3. EDITING STYLES:
   - All styles are in "style.css"
   - Each section is clearly labeled with comments
   - Modify padding, margins, font sizes, etc. as needed
   - The website uses CSS variables for consistency

4. MODIFYING JAVASCRIPT:
   - Open "script.js" to edit interactive features
   - Includes: mobile menu toggle, smooth scrolling, animations
   - Each function is documented for easy understanding

CUSTOMIZATION TIPS
--------------------------------------------------------------------------------
1. CHANGING THE LOGO:
   - Find ".logo" class in navigation section of each HTML file
   - Replace "Fira Code" text with your desired name

2. ADDING MORE PAGES:
   - Copy any existing HTML file
   - Rename it to your desired page name
   - Update the content inside <section> tags
   - Add a link to the new page in all navigation menus

3. MODIFYING FOOTER:
   - Footer is at the bottom of each HTML file
   - Update copyright year, links, and social media icons
   - To add more social icons, copy the existing GitHub icon structure

4. CHANGING FONTS:
   - In style.css, find the "font-family" property in the body selector
   - Replace with your preferred font family
   - For custom fonts, add @import or link in HTML <head> section

TROUBLESHOOTING
--------------------------------------------------------------------------------
Issue: Styles not loading
Solution: Make sure all files are in the same directory and refresh browser (Ctrl+F5)

Issue: Menu not working on mobile
Solution: Clear browser cache and ensure script.js is properly linked

Issue: Images or links not working
Solution: Check that file paths are correct and files are in the same folder

Issue: Layout looks broken
Solution: Ensure you're using a modern browser and JavaScript is enabled

DEPLOYMENT OPTIONS
--------------------------------------------------------------------------------

2. GITHUB PAGES:
