‚öôÔ∏è Setting Up FiraCode in VSCode

After installing the Fira Code font, follow these steps to enable it in Visual Studio Code:

1. Open VSCode Settings
Press Ctrl+, (Windows/Linux) or Cmd+, (macOS)

2. Search for "font family"
In the Editor: Font Family field, enter: Fira Code

3. Enable Font Ligatures
Search for "font ligatures" and check the box, or add to settings.json:
"editor.fontLigatures": true

4. Restart VSCode
Close and reopen VSCode to see the beautiful FiraCode Nerd Font with programming ligatures!
