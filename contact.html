Frequently Asked Questions

Q: How do I install Fira Code font?
A: Visit our Download page for installation instructions for all platforms.

Q: How do I enable ligatures in VSCode?
A: After installing FiraCode Nerd Font, add "editor.fontFamily": "Fira Code" and "editor.fontLigatures": true to your VSCode settings.

Q: Is Fira Code free?
A: Yes! Fira Code is completely free and open-source under the SIL Open Font License.
